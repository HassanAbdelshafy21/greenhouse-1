Circuit Diagram Image Location
================================

Please save your circuit diagram image as:
  circuit-diagram.png

in this directory (docs/images/).

The image should be the Cirkit Designer schematic showing:
- ESP32 microcontroller
- 4-channel relay module
- Stepper motor and driver
- DHT22 sensor
- Ultrasonic sensor
- pH sensor
- MQ135 air quality sensor
- LED strip
- Pumps and fans
- Power supply
- All wiring connections

This image is referenced in the README.md file in the Hardware Requirements section.
